Memo

A Reminding Android App

An android application is constructed to solve the issue of remembering everything. It has features where it's users can write important stuffs in the notes and in order to get reminded about any sort of work then they could use alarming feature.

Flutter framework and Dart programming language is used to complete this project. App consists of three tabs or pages- Home Page, Notes, Reminding Alarm to provide corresponding features to it's users. Firebase is used as a backend service to store the data provided in the notes section by the users. Gradient Effect is used to give better Graphic Interface to the App and splash service is also used.

In future various other features can be integrated like security system can be attached with the Notes so that certain private notes could be secured.
