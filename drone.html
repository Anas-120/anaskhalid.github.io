Sentinel Drone

A Nano Surveillance Drone

This project focuses on developing a nano Drone manipulated wirelessly, leverage them for surveillance tasks. Robot Operating System (ROS) and Gazebo was used for the wireless navigation of the drone over the particular area.

Using methodologies like image processing (using OpenCV) investigation of area is done and then suspected object's data so found is uploaded to GIS (Geographic Information Systems) for georeferencing so that it's geographic position could be ploted on a base map.
